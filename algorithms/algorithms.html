Algorithm Categories and Examples

Brute Force Algorithm

A Brute Force Algorithm is the straightforward approach i.e., the first approach that comes to our mind on seeing the problem.

More technically it is just like iterating every possibility available to solve that problem.

Example 1: find two numbers in an array that add up to a given target sum.

public static int[] findSum(int[] nums, int target) {
                            for (int i = 0; i < nums.length; i++) {
                                for (int j = i + 1; j < nums.length; j++) {
                                    if (nums[i] + nums[j] == target) {
                                        return new int[]{i, j}; // Return indices of the two numbers
                                    }
                                }
                            }
                            return null; // No solution found
                        }

Example 2: find all possible combinations of a given string (e.g., generating all permutations):

                        public static void permute(String str, String result) {
                            // Base case: if no characters left to permute, print the result
                            if (str.length() == 0) {
                                System.out.println(result);
                                return;
                            }

                            // Try each character as the next in permutation
                            for (int i = 0; i < str.length(); i++) {
                                // Get current character
                                char ch = str.charAt(i);
                                // Form the remaining string by excluding the current character
                                String remaining = str.substring(0, i) + str.substring(i + 1);
                                // Recurse with the remaining string and add current char to result
                                permute(remaining, result + ch);
                            }
                        }

                        public static void main(String[] args) {
                            String input = "ABC";
                            System.out.println("All permutations of " + input + ":");
                            permute(input, "");
                        }
Recursive Algorithm

A process in which a function calls itself directly or indirectly is called recursion and the corresponding function is called a recursive function.

A recursive algorithm takes one step toward solution and then recursively call itself to further move. The algorithm stops once we reach the solution.

Example: Factorial of a Number

public static int fact(int n) {
                            // BASE CONDITION
                            if (n == 0)
                                return 1;
                        
                            return n * fact(n - 1);
                        }

                        public static void main(String[] args) {
                            System.out.println("Factorial of 5 : " + fact(5));
                        }

Divide and Conquer Algorithm

A divide and conquer algorithm is a strategy for solving problems by breaking them into smaller subproblems, solving each subproblem independently, and then combining their solutions to solve the original problem.
It’s like tackling a big task by splitting it into manageable chunks, solving each chunk, and piecing the results together.

Example 1 Binary Search: 

 
                            public static int binarySearch(int[] arr, int target, int left, int right){
                                if(left>right){
                                    return -1;
                                }
                                int mid = left + (right-left)/2;
                                if(arr[mid]==target){
                                    return mid;
                                } else if(arr[mid]>target){
                                    return binarySearch(arr, target, left, mid-1);
                                } else{
                                    return binarySearch(arr,target,mid+1, right);
                                }
                            }

                            public static void main(String[] args){
                                int arr = [2,3,4,10,40,50,60,70];
                                int target = 10;
                                int result = binarySearch(arr, target, 0, arr.length - 1);
                                System.out.println("Element : "+ target + " found at index: "+ result);
                            }

Example 2 Merge Sort: 

 
                            public static void mergeSort(int[] arr, int left, int right){
                                if(left less than right){
                                    int mid = left + (right-left)/2;
                                    mergeSort(arr, left, mid);
                                    mergeSort(arr, mid + 1, right);
                                    merge(arr, left, mid, right);
                                }
                            }

                            private static void merge(int[] arr, int left, int mid, int right){
                                int n1 = mid - left + 1;
                                int n2 = right - mid;

                                int[] leftArr = new int[n1];
                                int[] rightArr = new int[n2];

                                for(int i=0;i less than n1;i++){
                                    leftArr[i] = arr[left+i];
                                }

                                for(int j=0;j less than n2;j++){
                                    rightArr[j] = arr[mid+1+j];
                                }

                                //merge temporary arrays

                                int i=0, j=0, k = left;

                                while(i less than n1 && j less than n2){
                                    if(leftArr[i]less than = rightArr[j]){
                                        arr[k++] = leftArr[i++];
                                    } else{
                                        arr[k++] = rightArr[j++];
                                    }
                                }
                                // Copy remaining elements
                                while (i less than n1) arr[k++] = leftArr[i++];
                                while (j less than n2) arr[k++] = rightArr[j++];
                            }
                            public static void main(String[] args) {
                                int[] arr = {64, 34, 25, 12, 22, 11, 90};
                                mergeSort(arr, 0, arr.length - 1);
                                System.out.println("Sorted array: " + java.util.Arrays.toString(arr));
                            }

Example 2: Quicksort:

 
                        function quickSort(arr, low, high) {
                            if (low lessthan high) {
                                // Partition the array and get the pivot index
                                let pi = partition(arr, low, high);
                                // Recursively sort the left and right partitions
                                quickSort(arr, low, pi - 1);
                                quickSort(arr, pi + 1, high);
                            }
                        }

                        function partition(arr, low, high) {
                            let pivot = arr[high]; // Choose the last element as pivot
                            let i = low - 1; // Index of smaller element

                            for (let j = low; j lessthan high; j++) {
                                // If current element is smaller than or equal to pivot
                                if (arr[j] lessthan = pivot) {
                                    i++;
                                    // Swap elements
                                    [arr[i], arr[j]] = [arr[j], arr[i]];
                                }
                            }
                            // Place pivot in its correct position
                            [arr[i + 1], arr[high]] = [arr[high], arr[i + 1]];
                            return i + 1; // Return pivot index
                        }

                        // Example usage
                        let arr = [64, 34, 25, 12, 22, 11, 90];
                        quickSort(arr, 0, arr.length - 1);
                        console.log("Sorted array:", arr);
