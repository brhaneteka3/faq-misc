Linux commands frequently asked questions and responses

Create a directory and change directory
  • mkdir directoryname
  • cd directoryname
  • cd ..
  • cd ./directoryname
  • pwd
  • ls
Create and open a file using linux commands?
  • touch testfilename.txt
  • echo "your text" > testfilename.txt
  • vim testfilename.txt
    • then press i to insert mode
    • When you are finished editing, press "esc", ":wq" and then "enter" to save and quit
  • cat testfilename.txt
