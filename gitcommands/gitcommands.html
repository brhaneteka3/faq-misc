Git commands frequently asked questions and responses

Check git version after installation
  • git --version
Navigate to the local directory created and initialize as a Git repository with initial branch called main in this case
  • git init -b main
Add updated files to staging area and configure git values
  • git add .
  • git config --global user.name "Your Name"
  • git config --global user.email "your.email@example.com"
  • git config user.name
  • git config user.email
After creating repository in Github with similar name, push changes using the following commands
  • git commit -m "first commit"
  • git branch -M main
  • git remote add origin https://github.com/githubuserId/myrepo.git
  • git push -u origin main
Changing branches, and pushing new changes to remote repository from local
  • git checkout -b branch-name : which is equivalent to
    • git branch branch-name
    • git checkout branch-name
  • git add .
  • git commit -m “commit message”
  • git push --set-upstream origin branch-name
    • Alternatively: git push -u origin separate-sections
